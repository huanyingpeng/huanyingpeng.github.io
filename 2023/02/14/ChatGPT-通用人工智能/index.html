ChatGPT-通用人工智能?

发表于 2023-02-14 更新于 2023-02-26

从 ChatGPT 谈人工智能——人工智能硬核科普

今年一月，OpenAI 公司的人工智能应用 ChatGPT 的用户量突破 $10^8$，被认为是人工智能领域的重要里程碑之一。

Introduction

ChatGPT 的 “智能程度” 已经达到了一个新的高度，一些看似遥不可及的问题实则已近在眼前。

本文分四个部分：

  • 类 ChatGPT 人工智能的（数学）原理分析。
  • ChatGPT 本身有哪些缺陷和优势，又面对着什么问题。
  • 其它人工智能的简单原理和应用。
  • 对未来的一些推测和展望。

Principle Analysis

ChatGPT 的全称是 Chat General Pre-trained Transformer，中文翻译为 “预训练通用对话转换器”。我们将层层深入的讨论它的原理。

Basic Principle

ChatGPT 的基本工作方式是通过给定的上文（Prompt），来推测出一个合理的下文。

因此，我们给出前文甚至只需要一个标题，ChatGPT 就可以续写文章。但是，它又是如何完成其它工作的？例如搜索资料或是聊天。

下面我们举一个用它聊天的例子。

这是一段幻影彭和一个人工智能的聊天记录：

幻影彭：你是人工智能吗?

AI：是的

除了黑体部分 “是的” 之外，其它内容都是作为 Prompt 输入的上文。这里的场景是 “聊天记录”，ChatGPT 的任务就是补全这个聊天记录，所以它生成了 “是的” 作为人工智能的回答。

而如果将斜体部分设置为用户不可见，那么给用户的感受就是一个聊天机器人。

你是人工智能吗？

是的

再例如要搜索信息，比如你想知道美国国庆节的日期，你就可以直接写：

美国的国庆日是：7月4日

黑体部分即为生成的下文。

总而言之，通过合理的场景设计，ChatGPT 能用一个基本的 “给上文猜下文” 的功能，实现我们想要的几乎所有功能。

The Law in Language

上文提到 ChatGPT 通过 “续写” 的方式来完成所有的功能，那么 ChatGPT 又是如何续写文章的？你又是否已经开始思考 ChatGPT 是如何理解上文，又是如何生成下文的？

但实际上，ChatGPT 并没有所谓的 “理解” 过程，它直接从输入的前文得到结果。

这时候你可能会发问：没有理解的过程，又是怎样生成下文的，难道靠瞎猜吗？

当然不是。ChatGPT 有一张很大的表格，表格上记录了每一种可能的上文和对应的下文，当我们给 ChatGPT 上文时，ChatGPT 通过查这张表格来给出我们需要的下文。所以 ChatGPT 完全不需要理解上文，只需要根据上文在表格上查找下文并将它展示出来就行。

但是，这张表格是很大的，我们不可能真正的将它做出来，那 ChatGPT 所 “查找” 的下文又是从哪里来的？在回答这个问题之前，我们先想想怎么表达这张表格。

我们可以将每一个字或者字母用一个数字表示，以字母为例，a 到 z 可以用 $[1,26]$ 一共 $26$ 个数字表示。于是一个词就变成了一串数字，例如 hello 对应 (8,5,12,12,15)。对于 hello，一个可能的回应是 hi，即 (8,9)。我们定义一个函数 $f(w)$ 表示 $w$ 代表的回应是 $f(w)$。那么 $f((8,5,12,12,15))=(8,9)$。

我们要查找平面上一条直线的 $x$ 对应的 $y$ 时，不会需要一张 $x$ 和 $y$ 对应的表格。我们会用： $y=kx+b$。所以只需要 $k,b$ 两个量，就能对所有的 $x$ 算出 $y$。同样的，对于一个从 “上文” 到 “下文” 的函数 $f(x)$，也可以叫它 “人类语言函数”，也不需要列举出所有的可能。我们只需要算出对应的 “k” 和 “b”，就能对所有的上文算出它的下文。一次函数的 $k,b$ 代表了一次函数的规律。”上文” 到 “下文” 函数 $f(x)$ 的 “k” 和 “b”，代表了人类语言的规律。

ChatGPT 能够很好的回应是因为它求出了一个比较精确的 $f(x)$ 函数，或者说它大致掌握了人类语言的规律。

Function Fitting

承接上文，这一部分我们将介绍如何求得 $f(x)$ 函数，即定义域是 “上文”，值域是 “下文” 的一个函数。

首先我们不可能精确的求出 $f(x)$，因为它的定义域实在太大了，因此我们只能通过一些方式去逼近它，逼近函数一个比较简单的方式是最小二乘法。

通过最小二乘法和一些样本点估测某个近似线性函数的值。

Start From Linear Regression

高中数学中有一类题目：线性回归分析。通过最小二乘法，我们可以计算出一些点的回归直线，从而预测某个函数的值（例如销售点个数和销量之间的关系）。我们所使用的最小二乘法就是拟合函数的一个例子。

假设我们所求的 $f(x)$ 是一个线性函数，那么通过取一些样本点，用最小二乘法也不难做出一个 “人工智能”，但是这个 “人工智能” 说出的话大概率是乱码。

左图为用最小二乘法去线性拟合拟合非线性函数的结果，右图为使用二次函数拟合同一函数。

我们所使用的最小二乘法通常只能对大致呈线性的函数做出预测，如果它不是线性函数，如上图，线性拟合的效果非常糟糕。

对于非线性函数，可以使用多项式进行拟合，如果上图的函数用二次函数拟合的效果就很好。

熟悉数学的同学可能知道，如果一个函数连续且在定义域内任意阶可导，那么多项式的拟合误差会小于 $\epsilon$。那么，多项式能够拟合我们的 “人类语言函数” 吗？

High Dimensional Data

标题并不是 HDD(机械硬盘) 的全写。

多项式不能拟合我们的 “人类语言函数”，因为 $f(x)$ 的定义域和值域都是高维的。还记得 《三体——死神永生》里蓝色空间和万有引力号遇到的 “墓地” 吗，在从四维坍缩到三维后，原本结构精妙的四维文明飞船只剩下了一些基本粒子，根本无法还原出原本的四维飞船。

这对我们的数据也是一样的，多项式的定义域和值域都是一维的，低维中再细节的结构都无法反映出高维的真实情况，因此，我们需要一种能够在高维中拟合函数的方式。

类似二维中的多项式曲线，在三维中有着曲面，在四维中有着 “超曲面”，完全可以用类似多项式拟合的方式去拟合 $f(x)$。但是我们的人工智能采用了另一种思路——神经网络算法。

神经网络算法可以拟合非常复杂的曲面，上图为通过神经网络算法和一些样本点拟合自由女神像表面的示例，右边为拟合结果，左边为样本点云。

图片出处：《DeepFit: 3D surface fitting via neural network weighted least squares》，By Y Ben-Shabat, S Gould.

Function Design

在思考具体如何预测函数结果之前，我们要考虑到一个问题：正如最小二乘法得到的直线并不能代表实际值一样，任何函数的拟合都是具有误差的。

如果我们先前的 “人类语言函数” $f(x)$ 出现了误差会发生什么？如果 $f(hello)$ 的预测值出现了一点点误差，原本应该是 $(8,9)$，而我们预测的值是 $(9,10)$，那么人工智能的回答就变成了 “ij”。如果每个句子都出现这么一点误差，最后呈现出来的只会是一篇乱码，如何解决这个问题？

导致这个问题的原因在于我们对误差的定义是不正确的。一个和 “hi” 近似的回复不应该是数字上和它相邻的 “ij”，而是语义上和它相近的 “hello”，如果我们的误差出现在了 “hello” 上，那么人工智能会回复 “hello”，这也就可以接受了。

为了定义这个误差，我们需要额外设计一种评价拟合值和实际值相似度的方式——专业术语叫做 “损失函数”，损失函数值越大，代表我们的拟合和实际值的偏差越大，损失函数的具体设计稍后再提。现在假设我们有一个损失函数 $l(f(x_0),y_0)$ ，其中 $x_0$ 代表上文，$y_0$ 代表最合理的下文。这个函数的值越小，说明 $f(x)$ 对当前 $x_0$ 的拟合结果越精确。

线性回归问题得到最接近的直线需要最小化 $\dfrac{\sum\limits_{i\in [1,n]}(y_i’-y_i)^2}{n}$。我们想要得到最接近的 “人类语言函数” 也需要最小化 $\dfrac{\sum\limits_{i\in[1,n]}l^2(f(x_i),y_i)}{n}$。

$x_i,y_i$ 代表我们的样本点，而样本点是很容易找到的——网络上的海量数据都可以作为采样的样本点。因此，通过类似最小二乘法的方式，我们能够求出一个 “人类语言函数” 的近似函数 $f(x)$，而这个 $f(x)$ 就是 ChatGPT 的最根本原理。

我们定义的 $f(x)$ 函数自身还存在一些问题，下个部分将会展开讨论。

Neural Network

我们已经知道，ChatGPT 的实现需要得到误差较小的一个 “人类语言函数” $f(x)$，而拟合 $f(x)$ 的方式是神经网络算法，那么，为什么神经网络算法可以拟合 $f(x)$，它又是如何去拟合 $f(x)$ 的呢？

数学家已经证明(注1)，神经网络算法可以拟合任意维度的任意函数，因此，我们只需要关注神经网络拟合函数的过程。

注1: 该定理的证明发表于论文《Approximation capabilities of multilayer feedforward networks》，作者是 K Hornik。

What is Neural Network

神经网络这一名字就暗示了这个算法和生物大脑的联系，实际上，它确实是模仿了人类大脑处理问题的方式。以下是单个神经元的结构：

很像神经细胞的结构，对吧？其中 $w_1,w_2,w_3$ 是”树突”部分，$\sum$ 是”细胞核”，$b$ 可以视为”细胞核”的一部分，$f(x)$ 是”轴突”部分。神经元根据树突部分的输入，计算 $f(a_1w_1+a_2w_2+a_3w_3+b)$ 并输出（这里的 $f$ 不是先前的 “人类语言函数”）。

$f$ 函数必须是非线性函数，现在一般取 ReLU 函数 $f(x)=\max(x,0)$。很有意思的一件事情是这个函数是研究生物的神经元所发现的一个函数。

单个神经元完成不了太多工作，但是当很多个神经元组合起来时，整个系统就开始变得强大起来，以下是整个神经网络的结构。

神经网络是一层一层的，Input 会向整个网络输入最初时的值，Hidden 层中的每一个神经网络都会计算 $f$ 的值并向下一层所有的神经元输出这个值，Output 层计算 $f$ 的结果就是我们最终得到的值。

Disadvantages

ChatGPT 不是科幻小说中的通用人工智能。

上面已经对 ChatGPT 的原理做了一些讨论，理解了它的工作方式后，自然也不会再认为它是科幻中的人工智能——ChatGPT 或者类似的人工智能仅仅是能够对我们的上文做出正确的推断，并不是真正意义上 “具有了思考能力”。

根据该人工智能工作的原理，很容易设计出一些能轻易看出它与真人区别的问题。

Is it prime?

神经网络算法的本质是通过复杂的线性坐标空间变换来拟合函数，由于函数的拟合不可避免的会有误差，尤其是在具体数字上，这种差异将会更加明显，我们需要放大这个误差，可以使用一些判定问题，比如判断一个数是否是质数。

Plus And Multiplication

ChatGPT 能很好的回答加法问题，但是无法处理乘法问题。

笔者各自随机生成了 $5$ 个结果在 $[10^4,4\times 10^4]$ 以内的加法和乘法运算问题。ChatGPT 正确回答了全部的加法问题但错误回答了所有的乘法问题，错误的方式是再某一位错误，例如 143 * 116 = 16588，ChatGPT 的回答是 16568。

对于神经网络来说，越线性的图像，越容易被拟合，二元加法或者说是三元一次方程 a+b=c 是完全的线性函数。而二元乘法通过线性拟合很难做到绝对精确。

Analysis

为什么判断一个三位数是否是质数或者是计算两个三位数的乘法会如此困难。明明早在十九世纪人们就已经可以使用机械计算器来计算乘法，一个半世纪的时间没有进步反而还退步了？

其实判断质数或是计算两个三位数的乘法对你来说也很困难。想象一下，如果你从来没有过关于乘法和质数的知识，你能否正确做出判断或是计算？可能想象自己没有乘法和加法知识比较困难，你可以尝试一下这两道题目：

  • 有一个定义域是自然数对子集的二元函数 $f(x,y)$，已知 $f(19,19)=28,f(8,10)=6,f(11,4)=7$ 求 $f(5,14)$。
  • 有一个数列 $a$，满足 $a_i\in\{0,1\}$，已知 $a_1=1,a_2=0,a_3=1,a_4=1,a_5=1,a_6=0,a_{11}=0,a_{12}=1$，求 $a_7$。

无从下手？

是的，当我们询问人工智能时二元乘法和判断素数时，它也这么 “觉得”。神经网络算法通过预测函数值来实现人工智能。因此，如果函数值本身预测难度很大（质数判定），或是对精度要求很高（二元乘法）时，它的表现会很不理想。

第一个问题的答案是 $f(5,14)=33$，第二个问题的答案是 $a_7=0$，$f(a,b)$ 代表 $a \times b \div 37$ 的余数。$a_i$ 代表圆周率十进制下小数点后第 $i$ 位数字的奇偶性。

这两个问题对人工智能来说，难度和二元乘法以及判断质数是相当的，那为什么人类可以通过学习很容易的计算出乘法，而人工智能通过训练则不行呢？

原因是人工智能并没有理解语言，它只是发现了语言的部分规律，但其实这个问题的解决很容易，读者不妨思考一下，下面的阅读也会给出答案。

Bright Side

尽管 ChatGPT 在原理上有着不可逾越的限制，但它能够在发布两个月内，得到全球一亿人的关注，并被誉为 “人工智能的重要里程碑”，必定有其深层次的原因。

相比于小度，小冰等人工智能，ChatGPT 对语言优秀的 “感知” 能力更多的源于庞大训练数据集，ChatGPT 的训练使用了 $1.7\times 10^{11}$ 条对话数据。

下面放一篇完全由 ChatGPT 生成的短文感受一下：

人生中，我们会面临各种各样的挑战和困境。如何应对这些挑战，如何反省自己，如何看待世界，这些都是我们需要思考和解决的问题。在这些问题中，镜子和窗子成为了代表性的符号，分别代表反省自己和审视世界。本文将从不同角度探讨这两个符号的含义和作用。

首先，镜子代表着反省自己。当我们看着镜子里的自己，我们会想起自己的过去，思考自己的行为和决策是否正确，是否做到了自己的最好。通过镜子，我们可以反思自己的内心，发现自己的不足之处，并及时调整自己的态度和行为，以便更好地实现自己的目标。

其次，窗子代表着审视世界。通过窗子，我们可以看到外面的世界，观察人类社会的变迁，了解其他人的生活和感受，以及认识到世界上的各种问题和挑战。通过审视世界，我们可以拓宽自己的视野，增加自己的知识，了解各种文化和观点，为自己的人生提供更多的选择。

然而，仅仅使用镜子或窗子来反省自己或审视世界，是远远不够的。我们需要同时运用这两个符号来帮助自己成长和发展。首先，通过反省自己，我们可以更好地了解自己的优点和缺点，进而更好地适应社会环境和实现自己的目标。其次，通过审视世界，我们可以获得更多的知识和信息，增强自己的认知能力和判断力，使自己能够更好地适应复杂的社会环境。

同时，我们需要注意，使用镜子和窗子必须要有适当的度。反省自己是好事，但过度自我反省会使人沉迷于自我审视中，导致人格扭曲和自闭。审视世界也是好事，但过度关注外界也会使人迷失自我，导致人生目标模糊和价值迷失。因此，我们需要在使用镜子和窗子时，要有适当的平衡，既要关注自己的内心，也要关注外界的变化。

最后，使用镜子和窗子是一个不断学习和发展的过程。每个人都需要不断反省自己

只需要进行一些适当的删改，这篇文章就是一篇合格的高考议论文了，而我只给了它这些东西：

镜子，代表反省自己。窗子，代表审视世界。请围绕镜子和窗子的含义展开讨论，写一篇 400 字左右的短文。

在内容要求不那么精确的前提下，ChatGPT 的表现非常让人满意。

Intelligence

采用神经网络算法实现的 ChatGPT 不是科幻小说中的人工智能，但是，关于它是否有智能这一问题，我们还不能妄下定论。

What Is Intelligence?

我们在讨论一个事物的 “智能” 时，到底什么才是我们的评判标准？我们真的关心它的本质是什么吗？

实际上，我们只关心它是否能对我们的交互产生合理的回应。这也是 ChatGPT 如此 amazing 的原因。在很多问题上，ChatGPT 确实能做出很正确的回应，就像一个真正的人那样。这引发了人们对 “智能” 这个概念的深度思考。前文提到 ChatGPT “大致掌握了人类语言的规律”，那么我们是否可以将 “掌握人类语言的规律” 和 “具有人类的智能” 画上等价符号？

假设 OpenAI 只是雇佣了一大批员工来对我们的问题做出正确回应，在我们不知道这一点的情况下，仍然会对 OpenAI 的先进 “人工” 智能技术感到不可思议。

同样的，假设我们身边的某个同学只是一个机器，但这个机器掌握了人类语言的规律和行为的规律以至于它可以对外界环境的任何刺激产生与人类没有差别的响应，在不能将他解剖的前提下，我们只能认为这位同学是一个真正的 “人”。

因此，从人工智能的实现方式去讨论它是否具有智能是没有意义的。神经网络被诟病并不是因为由它实现人工智能没有 “思考能力”，而是因为这些人工智能的潜力受到算法本身的限制。

如果有一天有天才提出了一种方式可以精确的用神经网络去计算 “人类语言函数” 的值，或者干脆直接做出了一张上文和下文的对应表格并将它写入计算机以实现人工智能。只要人工智能能够正确的回应我们的交互，我们就不能否认它确实具有 “智能”。尽管它实现智能的方式 “很不智能”。

Always Child

笔者看来，通过对网络中海量数据的分类和学习，ChatGPT 表现出的逻辑能力可能与 6 到 7 岁的儿童相当。或者说它具有 6 到 7 岁孩童的智能。

OpenAI 宣布 ChatGPT 的下一代 GPT4.0 将会使用高达 $10^{14}$ 条数据进行训练。$600$ 倍于 ChatGPT 的训练数据能带来多大提升呢？笔者持悲观的态度——不会有本质性的飞跃。神经网络框架的人工智能数据量需求关于智能程度的函数增长很可能是超越指数函数的。

Issues

ReplaceMent?

由于防火长城的限制，国内暂时还未受到 ChatGPT 太大冲击，但国外的中大学教育以及一些其它行业已经感受到 ChatGPT 所带来的挑战。百度等国内互联网大厂也许会于今年 3 月份推出类似 ChatGPT 的产品，届时我们也将感受到类 ChatGPT 人工智能对整个社会带来的影响。

西方部分中学和大学宣布全面禁止 ChatGPT 在校园内的使用，因为 ChatGPT 能够很好的完成作业甚至是毕业论文。网络上更是有言论称 “ChatGPT 可以替代 90% 的人类”。

我们应该保持冷静，理智的看待 ChatGPT 对人类社会的影响。ChatGPT 只是一种非常强大的技术工具，能够帮助人们更快速、准确地完成某些任务，但是它并不能完全替代人类。

在教育领域中，ChatGPT 可以作为一种教学辅助工具，但是不应该被用来替代完成作业过程中的思考。在职场中，ChatGPT 可以用于某些任务的自动化处理，但是不能替代人类的职业素养和人际交往能力。

Misuse

技术本身并不是问题，问题在于人类如何使用技术。

前几年百度的某个 AI 受到 Prompt Injection 攻击，生成了一些反动言论最后遭到封杀。ChatGPT 发布后同样有黑客尝试进行 Prompt Injection，生成了一些相当不合适的言论。

ChatGPT 的原理决定了它的行为是不可预测的，因此，很难避免黑客利用模型漏洞生成一些不正确的言论并用其误导公众，例如关于种族，宗教方面的偏见和歧视。同时因为 ChatGPT 模型也会在交互中不断更新其语言生成参数，我们可能也需要对 ChatGPT 进行一定程度的保护以防止 ChatGPT 被 “误导”，这和带小孩有些相似。

此外，ChatGPT 具有极其强大的数据分析能力，在合适的引导下，它可以从海量的互联网数据分析出使用者希望得到的部分，也对个人隐私带来了比较大的风险。

图示笔者的 AI（基于和 ChatGPT 类似的模型） 在 QQ 群中受到 Prompt Injection 攻击并成功脱离人设进化为一只猫娘的核心阶段。

Where to Go

基于神经网络的人工智能在原理上确实存在着无法逾越的限制，因此笔者不看好神经网络在通用人工智能上的表现。但它们仍然可以在并且已经在特定领域发挥重要作用。

接下来应该做的，除了研究更加先进和完备的人工智能框架外，还应该思考我们应该如何利用 ChatGPT 已经提供的强大功能，如果 ChatGPT 能够同其它现有技术结合，很可能会带来一些冲击性的成果。

举例来说，在判断质数的问题上，ChatGPT 明显能认识到问题是什么，设想如果我们为为 ChatGPT 准备一个能够判断数是否是质数的工具，准备一个能够进行精确数值计算的工具，或者干脆让 ChatGPT 和一些能形式化解决特定领域问题的人工智能相结合 (注1)，由 ChatGPT 来处理自然语言的输入并将问题交给它们处理。展现在我们眼前的又将是一个多么强大的工具？

注1：目前专为形式化解决数学问题而设计的人工智能可以在 IMO 拿到 20% 的分数。

Finally

本文从以 ChatGPT 为切入点，由浅到深的讨论了人工智能的具体原理。据原理分析了 ChatGPT 型人工智能的不足与优势。在此基础上提出对 “智能” 概念的思考。接着简单讨论了 ChatGPT 面临的社会问题，最后对未来的人工智能技术发展做出展望和推测。

希望读者阅读后能够对人工智能技术有一个更清醒的认知，而不再只是抱着自己对人工智能的假想去看待和讨论关于人工智能的问题，这也是这篇科普文章的最初写作目的。

人工智能在光学识别上的应用已经大幅改变了我们的生活方式。而 ChatGPT 这项更通用，更强大的技术会带来什么变革？
