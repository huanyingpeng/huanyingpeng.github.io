Basic Auth

Basic Auth 是一种在头部提供验证账户和密码的方式，具体的，需要在 headers 参数中加入 Authorization 字端，其值为 [验证方式] [Base64 编码的账号和密码]。

参考

部分验证会忽略 Base64 的 =，所以爆破前先检查真实方式是否有 =。
